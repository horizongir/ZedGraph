    Class ErrorBar

    This class handles the drawing of the curve ErrorBar objects. The Error Bars are the vertical lines with a symbol at each end.

public class ErrorBar : ICloneable, ISerializable
    Remarks

    To draw "I-Beam" bars, the symbol type defaults to HDash, which is just a horizontal line. If BarBase is Y-oriented, then the symbol type should be set to VDash to get the same effect.

    boolean value that indicates if the "base" axis for this ErrorBar is the X axis. True for an XAxis base, false for a YAxis or Y2Axis base.

    The dependent axis position of the bottom of the error bar in pixel units
    Single scaleFactor

    The scaling factor for the features of the graph based on the BaseDimension. This scaling factor is calculated by the CalcScaleFactor() method. The scale factor represents a linear multiple to be applied to font sizes, symbol sizes, etc.

    A pen with attributes of Color and PenWidth for this ErrorBar
    Boolean isSelected

    Indicates that the ErrorBar should be drawn with attributes from the Selection class.
    PointPair dataValue

    The data value to be used for a value-based color gradient. This is only applicable for GradientByX, GradientByY or GradientByZ.
